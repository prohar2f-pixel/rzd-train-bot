# РЖД Бот - Поиск Билетов

Бот для автоматического поиска билетов на поезд Москва → Симферополь 13-15 июля 2026 в Telegram.

## Быстрый старт

### 1. Подготовка Telegram

1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте `/newbot`
3. Назовите бота: `RZD Train Bot` (или как угодно)
4. Назовите username: `rzd_train_bot_<ваше_имя>` (должно быть уникальным)
5. Скопируйте **Token** (выглядит как `123456789:ABCdefGHIjklmnoPQRstuvWXYZ`)

### 2. Найти ваш Chat ID

1. Откройте найденного бота (или начните диалог)
2. Напишите в чат: `/start`
3. Перейдите сюда: https://api.telegram.org/bot**YOUR_TOKEN**/getUpdates (вместо YOUR_TOKEN вставьте ваш токен)
4. Найдите `"id": XXXXXXX` в JSON - это ваш Chat ID

### 3. Заполнить .env

```bash
# Скопируйте .env.example в .env
cp .env.example .env
```

Отредактируйте `.env`:
```
TELEGRAM_TOKEN=123456789:ABCdefGHIjklmnoPQRstuvWXYZ
TELEGRAM_CHAT_ID=987654321
```

### 4. Установить зависимости

```bash
npm install
```

### 5. Запустить бот

```bash
npm start
```

Если всё работает, вы получите сообщение в Telegram с подтверждением 🎉

## Как это работает?

- **Каждые 10 минут** бот проверяет сайт РЖД
- **При появлении билетов** отправляет уведомление в Telegram
- **Ищет**: 3 места в плацкарте (1 нижнее обязательно, 2 рядом)
- **Сроки**: 13-15 июля 2026, должны приехать в Симферополь к 17 июля до 10:00

## Деплой на Railway

1. Создайте репозиторий на GitHub
2. Залейте код
3. На [railway.app](https://railway.app) нажмите "New Project"
4. Выберите "Deploy from GitHub"
5. Выберите репозиторий
6. В переменных окружения добавьте:
   ```
   TELEGRAM_TOKEN=ваш_токен
   TELEGRAM_CHAT_ID=ваш_id
   ```
7. Деплой начнётся автоматически

## Troubleshooting

### "TELEGRAM_TOKEN не установлен"
→ Проверьте файл `.env` в корне проекта, он должен содержать `TELEGRAM_TOKEN=...`

### "Не получаю уведомления"
→ Проверьте Chat ID через API (см. выше)

### "Браузер не запускается на Railway"
→ Это нормально, он будет использовать API вместо парсинга

### "API РЖД не доступен"
→ Бот автоматически переключится на парсинг сайта
